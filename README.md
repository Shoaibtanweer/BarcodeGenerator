# Barcode Generator 

## Features
- Supports EAN-13, UPC-A, EAN-8, ITF, Code 39, Code 128, and QR Code
- Instant preview
- Download barcodes as PNG
- Print directly
- Light/Dark mode

## How to Use
1. Choose barcode type
2. Enter one or multiple lines of data
3. Click "Generate"
4. Download or print your barcodes
